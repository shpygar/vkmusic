Тестовое задание

Вводная
Есть сервис - VKontakte. У него есть SDK для iOS, подробное описание которого
находится по адресу https://vk.com/dev/ios_sdk. В качестве тестового задания
предлагается написать клиент для сервиса музыки со следующим
функционалом:
1. Возможность залогиниться/разлогиниться пользователем
2. Возможность посмотреть музыку пользователя. Клиент должен уметь делать
обновление.
3. При выборе элемента в ленте запускать онлайн плеер.


Уточнения
•Дизайн любой, какой хотите, ничего изысканного придумывать не надо.
Стандартные контролы всех устроят.
•Догрузка элементов в список должна делаться с использованием pull-to-refresh.
•Проект должен вестись в git. Коммиты должны быть с понятными комментариями.
•С ARC
•Сделать кеширование всего этого дела в CoreData.

Это задание делал в качестве этапа реализации более сложного приложения для прослушивании аудиокниг в оффлайне с возможностью запоминания места остановки.
